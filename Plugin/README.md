# unity-unity-pathtracer-plugin
Wrapper for tinybvh.

Derived from https://github.com/andr3wmac/unity-tinybvh.

### Windows
```
mkdir build
cd build
cmake ..
cmake --build . --config Release
cmake --install .
```

## macOS
```
mkdir build
cd build
cmake -G Xcode ..
cmake --build . --config Release
cmake --install .
```
